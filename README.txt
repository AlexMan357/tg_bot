1- Бот запрашивает данные с сайта stoleshka.ru и выводит запросы по командам.
2 - Записывает результаты запроса о скидках(req) в  базу данныx history (поля: дата(date_field),
наименование искусственного камня(title),
ссылка на картинку с описанием(image),
текущая цена(price_current),
старая цена(price_old),
размер погонного метра(size))

Команды:
   /start, /help - выводит ссылку для перехода на сайт, выводит перечисление возможных команд

   /req - выполняет запрос к сайту. Выбирает позиции искусственного камня, на который скидка не менее 50%
          Запрос выводится в виде: наименование камня, ссылка на картинку с описанием, текущая цена
          Результат запроса записывается в базу данных.
          Если в одну дату выполняется несколько запросов, то в базу данных записывается последний
          выполненный запрос.
          Получает данные с первой страницы сайта.

   /next - выполнить запрос к следующей странице сайта

   /history - выполняет чтение полей базы данных history и выводит историю запроса о скидках
          за последние три дня: дата, наименование камня, цена на момент скидки

   /clearhistory - выполняет удаление всех записей базы данных

   /allbrands - выполняет запрос к сайту и выводит информацию о брендах, представленных на сайте
   в формате: наименование бренда - ссылка на список позиций бренда

   /contacts - выполняется запрос контактов организации, представляющей сайт и выводится результат.
